Features

Utility Commands

Keep your server organized with powerful moderation tools, user info lookup, server stats, polls, and announcements.

Fun Mini-Games

Entertain your members with interactive games like trivia, hangman, word scramble, and number guessing games.

Moderation Tools

Maintain server order with ban, kick, timeout, warning system, and auto-moderation features to filter unwanted content.

Customization

Configure Float to match your server's specific needs with easy-to-use settings and preference options.

Commands

poll

/poll [question] [options]

Create interactive polls for your server members to vote on.

serverinfo

/serverinfo

Display detailed information about your Discord server.

userinfo

/userinfo [user]

Show profile details about a specific server member.

embed

/embed [title] [description]

Create custom embedded messages with rich formatting.
